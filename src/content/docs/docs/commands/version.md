---
title: version
description: Print CLI version information.
---

## Synopsis

```
flowmi version
```

## Description

Print version information for the Flowmi CLI. Output includes the version number, Git commit hash, build date, Go version, and OS/architecture.

## Flags

This command has no flags.

## Examples

```bash
# Print version info
flowmi version

# Get version as JSON (useful for bug reports or scripting)
flowmi version --json
```
