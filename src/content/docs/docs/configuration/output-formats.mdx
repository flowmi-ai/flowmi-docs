Flowmi CLI outputs human-readable text by default. Pass the `--json` global flag to get machine-readable JSON instead.

## Formats

### text (default)

Human-readable, styled output. This is the default.

### json

Machine-readable JSON, ideal for scripting and piping to tools like `jq`. Enabled with the `--json` flag.

## Comparison

The same command — `flowmi note list` — in each format:

## Scripting with JSON

Use `--json` together with `jq` to extract and transform data:

```sh
# Get IDs only
flowmi note list --json | jq '.[].id'

# Count notes
flowmi note list --json | jq 'length'
```

## Help format

The `--format json` flag is a separate option that outputs help text as JSON. This is distinct from the `--json` flag which controls command result formatting.
