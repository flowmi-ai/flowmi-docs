Flowmi CLI supports three output formats controlled by the `--output` (or `-o`) flag: **text**, **json**, and **table**.

## Formats

### text (default)

Human-readable, styled output. This is the default when no `--output` flag is provided.

### json

Machine-readable JSON, ideal for scripting and piping to tools like `jq`.

### table

Tabular format with aligned columns.

The same command — `flowmi note list` — in each format:

## Scripting with JSON

Use the `json` format together with `jq` to extract and transform data:

```sh
# Get IDs only
flowmi note list -o json | jq '.[].id'

# Count notes
flowmi note list -o json | jq 'length'
```

## Help format

The `--format json` flag is a separate option that outputs help text as JSON. This is distinct from the `--output` flag which controls command result formatting.
