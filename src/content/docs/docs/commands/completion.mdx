## Synopsis

```
flowmi completion <shell>
```

## Description

Generate shell completion scripts for the Flowmi CLI. Completions provide tab-completion for commands, subcommands, and flags. Supported shells: **bash**, **zsh**, **fish**, and **powershell**.

## Installation

<Tabs>
  <TabItem label="Bash">
    Load completions in your current session:

    ```bash
    source <(flowmi completion bash)
    ```

    To load completions automatically on every new session, add the line to your `~/.bashrc`:

    ```bash
    echo 'source <(flowmi completion bash)' >> ~/.bashrc
    ```
  </TabItem>

  <TabItem label="Zsh">
    Generate the completion file and place it in your `fpath`:

    ```zsh
    flowmi completion zsh > "${fpath[1]}/_flowmi"
    ```

    Then reload completions:

    ```zsh
    compinit
    ```

    You may need to start a new shell session for the changes to take effect.
  </TabItem>

  <TabItem label="Fish">
    Load completions in your current session:

    ```fish
    flowmi completion fish | source
    ```

    To load completions automatically on every new session, save them to the completions directory:

    ```fish
    flowmi completion fish > ~/.config/fish/completions/flowmi.fish
    ```
  </TabItem>

  <TabItem label="PowerShell">
    Load completions in your current session:

    ```powershell
    flowmi completion powershell | Out-String | Invoke-Expression
    ```

    To load completions automatically, add the above line to your PowerShell profile.
  </TabItem>
</Tabs>
