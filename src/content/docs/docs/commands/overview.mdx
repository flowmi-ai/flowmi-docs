## Global flags

These flags are available on every command.

| Flag | Short | Description | Default |
|------|-------|-------------|---------|
| `--config` | | Path to config file | `~/.config/flowmi/config.toml` |
| `--json` | | Output in JSON format | |
| `--debug` | | Enable debug logging (HTTP requests/responses) | |
| `--format` | | Help format: `text`, `json` | `text` |
| `--help` | `-h` | Show help | |

## Output formats

The `--json` flag switches output from human-readable text to machine-readable JSON.

- **text** (default) — Human-readable output designed for interactive use.
- **json** — Machine-readable JSON output. Useful for piping into `jq`, scripting, or integrating with other tools.

```bash
# Default human-readable output
flowmi note list

# JSON for scripting
flowmi note list --json
```

## Aliases

Most `list` subcommands accept `ls` as a shorter alias. For example, `flowmi note ls` is equivalent to `flowmi note list`.
